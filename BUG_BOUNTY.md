# Bug Bounty Program

## Scope

- All smart contracts, protocol logic, and critical infrastructure.
- Excludes front-end UI bugs unless they impact security.

## Rewards

- Severity-based payouts (Critical, High, Medium, Low).
- Paid in USDC or native token.

## Submission Process

- Submit via Immunefi, HackenProof, or email [security@aetheron.org](mailto:security@aetheron.org).
- Include detailed report, PoC, and remediation suggestions.

## Responsible Disclosure

- No public disclosure until fix is deployed.
- Coordinated communication with security team.
